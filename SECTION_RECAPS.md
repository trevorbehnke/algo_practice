# Section 1

## You just completed all 5 problems! Here's a recap of the patterns you learned:

1. Build and return — construct a result in a loop, return it after (reverse string)
2. Track and update — maintain a running value with conditional updates (find max)
3. Early return on failure — bail out immediately when something's wrong (palindrome)
4. Frequency map — use an object to count occurrences (count characters)
5. Hash map lookup — store seen values for O(n) lookups instead of nested loops (two sum)
